
A responsive and accessible Profile Card built using semantic HTML, CSS, and vanilla JavaScript.  
This project was completed as part of the Frontend Wizards Stage 1B task.

 Features

- Fully responsive profile card (mobile, tablet, desktop)
- Semantic HTML structure (`article`, `header`, `section`, `nav`, `figure`)
- Accessible design (alt text, focus states, aria-live time updates)
- Live epoch time display (milliseconds since Unix epoch)
- Social media links open in a new tab safely
- Separate sections for hobbies and dislikes
- All required `data-testid` attributes included for testing

 Technologies Used

- HTML5
- CSS3 (Flexbox + Media Queries)
- Vanilla JavaScript (DOM manipulation)
